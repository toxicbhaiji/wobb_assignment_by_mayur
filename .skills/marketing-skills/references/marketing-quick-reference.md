# Marketing Skills Quick Reference

## Copywriting Frameworks

### AIDA (Attention → Interest → Desire → Action)
Best for: Ads, landing pages, sales pages

```
Attention: Grab attention with a bold headline or hook
Interest: Build interest with relevant details and benefits
Desire: Create desire with emotional triggers and social proof
Action: Call to action with clear next steps
```

### PAS (Problem → Agitate → Solve)
Best for: Pain-point pages, problem-aware audiences

```
Problem: Identify the painful problem
Agitate: Make the pain feel urgent and real
Solve: Present your solution as the relief
```

### FAB (Features → Advantages → Benefits)
Best for: Product pages, feature announcements

```
Features: What it is (the facts)
Advantages: What it does better than alternatives
Benefits: What the user gets (the outcome)
```

### StoryBrand
Best for: Homepage, about page, brand storytelling

```
Character: The customer is the hero
Problem: They have a problem
Guide: Your brand is the guide
Plan: You have a plan
Call to Action: Invite them to take action
Success: Show what success looks like
Failure: Show what failure looks like (optional)
```

## SEO Quick Wins Checklist

### On-Page SEO
- [ ] Title tag: Under 60 chars, primary keyword first
- [ ] Meta description: Under 160 chars, include CTA
- [ ] H1: One per page, matches search intent
- [ ] H2-H6: Logical hierarchy with keywords
- [ ] URL: Short, descriptive, keyword-rich
- [ ] Internal links: 2-5 per page, relevant anchor text
- [ ] Images: Alt text, descriptive filenames, compressed
- [ ] Schema markup: Appropriate structured data

### Technical SEO
- [ ] Page speed: Under 3 seconds (LCP)
- [ ] Mobile-friendly: Responsive design
- [ ] HTTPS: SSL certificate active
- [ ] XML sitemap: Submitted to Google
- [ ] Robots.txt: Properly configured
- [ ] Canonical tags: Prevent duplicate content
- [ ] 404 handling: Custom 404 page, redirects
- [ ] Core Web Vitals: All metrics in "Good" range

### Content SEO
- [ ] Keyword research: Search volume and intent analyzed
- [ ] Content depth: Comprehensive coverage of topic
- [ ] Freshness: Regularly updated content
- [ ] E-E-A-T: Expertise, Experience, Authority, Trust
- [ ] Featured snippets: Structured for position zero
- [ ] Internal linking: Topic clusters and pillar pages

## CRO (Conversion Rate Optimization) Checklist

### Landing Page Elements
- [ ] **Headline**: Clear, benefit-driven, matches ad/source
- [ ] **Subheadline**: Supports headline, adds specificity
- [ ] **Hero image/video**: Relevant, high-quality, shows product
- [ ] **CTA button**: Contrasting color, action-oriented text
- [ ] **Form**: Minimal fields, clear labels, inline validation
- [ ] **Social proof**: Testimonials, logos, user counts, ratings
- [ ] **Trust signals**: Security badges, guarantees, certifications
- [ ] **Risk reversal**: Free trial, money-back guarantee, no credit card
- [ ] **Urgency/scarcity**: Limited time, limited spots, countdown timer
- [ ] **FAQ**: Addresses common objections

### Page Design
- [ ] Single focus: One primary goal per page
- [ ] Visual hierarchy: Most important elements first
- [ ] White space: Not cluttered, breathing room
- [ ] Mobile optimized: Touch-friendly, readable
- [ ] Fast loading: Under 3 seconds
- [ ] Accessible: WCAG 2.1 AA compliant

### Psychology Principles
- [ ] **Reciprocity**: Give value before asking (free tool, content)
- [ ] **Social proof**: Show others are using/buying
- [ ] **Authority**: Expert endorsements, certifications
- [ ] **Scarcity**: Limited availability
- [ ] **Urgency**: Time-limited offers
- [ ] **Loss aversion**: Highlight what they lose by not acting
- [ ] **Anchoring**: Show higher price first, then discount
- [ ] **Decoy effect**: Add a decoy option to make target more attractive

## Email Marketing Frameworks

### Welcome Sequence (5-7 emails)
1. **Welcome**: Deliver lead magnet, set expectations
2. **Story**: Share origin story, build connection
3. **Education**: Teach something valuable
4. **Social proof**: Case studies, testimonials
5. **Soft sell**: Introduce product, focus on problem
6. **Hard sell**: Direct offer, urgency/scarcity
7. **Objection handling**: Address common objections

### Abandoned Cart Sequence (3 emails)
1. **Reminder**: Friendly reminder, show cart contents
2. **Incentive**: Small discount or free shipping
3. **Urgency**: Last chance, cart expires soon

### Re-engagement Sequence (3-4 emails)
1. **We miss you**: Acknowledge absence, ask for feedback
2. **Value reminder**: Share best content or updates
3. **Incentive**: Special offer to come back
4. **Goodbye**: Unsubscribe if no response

## Ad Creative Best Practices

### Headlines
- Lead with the benefit, not the feature
- Use numbers and specificity
- Ask a question that resonates
- Create curiosity gap (but deliver in ad)
- Use power words: Free, New, Proven, Instant, Exclusive

### Images
- Show the product in use (not just the product)
- Include people using the product
- Use contrasting colors to stand out
- Keep text under 20% of image (Meta rule)
- Test lifestyle vs. product shots
- Use video for complex products

### Calls to Action
- Use action verbs: Get, Start, Try, Claim, Access
- Be specific: "Get My Free Guide" vs "Submit"
- Create urgency: "Start Your Free Trial Now"
- Reduce friction: "No Credit Card Required"

## A/B Testing Framework

### What to Test
**High Impact:**
- Headline and value proposition
- CTA button text and color
- Pricing and plans
- Form length and fields
- Social proof placement

**Medium Impact:**
- Hero image/video
- Page layout and structure
- Color scheme
- Navigation structure

**Low Impact:**
- Font choices
- Minor copy changes
- Button shape
- Spacing adjustments

### Testing Process
1. **Hypothesis**: "Changing X to Y will increase Z because..."
2. **Prioritize**: Use ICE score (Impact × Confidence × Ease)
3. **Design**: Create variations, ensure statistical validity
4. **Run**: Minimum 1 week, 100+ conversions per variant
5. **Analyze**: Check statistical significance (p < 0.05)
6. **Implement**: Winner becomes control, iterate
7. **Document**: Record learnings in testing log

## Growth Metrics Framework

### AARRR (Pirate Metrics)
- **Acquisition**: Where do users come from? (traffic, signups)
- **Activation**: Do they have a great first experience? (Aha moment)
- **Retention**: Do they come back? (DAU/MAU, churn rate)
- **Revenue**: How do you make money? (ARPU, MRR, LTV)
- **Referral**: Do they tell others? (NPS, viral coefficient)

### North Star Metric
Choose one metric that best captures core value:
- SaaS: Weekly active teams, features used
- E-commerce: Monthly purchases, repeat purchase rate
- Content: Daily active readers, time spent
- Marketplace: Transactions per user, GMV

## Customer Research Methods

### Quantitative
- **Surveys**: NPS, CSAT, product-market fit survey
- **Analytics**: Funnel analysis, cohort analysis, path analysis
- **A/B Tests**: Preference tests, pricing tests

### Qualitative
- **Interviews**: Jobs-to-be-done, problem discovery
- **Usability Tests**: Task completion, think-aloud protocol
- **Support Tickets**: Common complaints and questions
- **Reviews**: App store, G2, Capterra, Trustpilot

### Research Questions
- What triggered you to look for a solution?
- What alternatives did you consider?
- What almost stopped you from signing up?
- What would you use if our product didn't exist?
- What's the main benefit you've received?
- How would you describe our product to a friend?

## Competitor Analysis Framework

### Direct Competitors
- Same category, same audience, same problem
- Analysis: Pricing, features, positioning, weaknesses

### Indirect Competitors
- Different category, same audience, same problem
- Analysis: Alternative solutions, switching costs

### Aspirational Competitors
- Larger companies in adjacent spaces
- Analysis: Best practices, messaging, growth tactics

### Comparison Page Structure
```
1. Acknowledge competitor is good (builds trust)
2. Highlight key differences (your strengths)
3. Address who each solution is best for
4. Include feature comparison table
5. Add social proof and testimonials
6. Clear CTA to try your solution
```

## Pricing Strategy Frameworks

### Pricing Models
- **Flat-rate**: Simple, predictable (e.g., $99/month)
- **Tiered**: Good-better-best structure
- **Usage-based**: Pay for what you use (e.g., API calls)
- **Per-seat**: Per user pricing (common in B2B SaaS)
- **Freemium**: Free tier + paid upgrades

### Pricing Psychology
- **Charm pricing**: $99 vs $100 (left-digit effect)
- **Anchoring**: Show highest price first
- **Decoy effect**: Add a decoy to make target attractive
- **Bundle pricing**: Combine products for perceived value
- **Free trial**: Reduce risk, increase adoption
- **Money-back guarantee**: Remove purchase anxiety

### Pricing Page Best Practices
- [ ] Clear plan comparison
- [ ] Most popular plan highlighted
- [ ] Annual discount visible
- [ ] FAQ addressing common questions
- [ ] Enterprise/contact sales option
- [ ] No hidden fees mentioned
- [ ] Currency localization (if applicable)
