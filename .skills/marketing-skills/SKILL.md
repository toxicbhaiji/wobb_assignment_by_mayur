# Marketing Skills

## Overview

A comprehensive collection of marketing skills for AI agents. Covers conversion optimization, copywriting, SEO, paid advertising, analytics, growth engineering, and sales enablement. Built for technical marketers and founders who want AI coding agents to help with marketing tasks.

## When to Use

- Optimizing landing pages, signup flows, or onboarding for conversions
- Writing or improving marketing copy (homepage, landing pages, emails, ads)
- Planning SEO strategy, content strategy, or programmatic SEO
- Creating and optimizing paid advertising campaigns
- Setting up analytics tracking and measurement
- Designing pricing, offers, or paywalls
- Planning product launches or feature announcements
- Building referral programs or community marketing
- Conducting customer research or competitor analysis

## Core Workflows

### Landing Page Optimization Workflow

```
1. AUDIT → Analyze current page (cro skill)
2. RESEARCH → Understand audience (customer-research skill)
3. COPY → Write optimized copy (copywriting skill)
4. DESIGN → Create visuals (image skill)
5. TEST → Set up A/B test (ab-testing skill)
6. MEASURE → Track conversions (analytics skill)
```

### SEO Content Workflow

```
1. STRATEGY → Plan topics (content-strategy skill)
2. RESEARCH → Analyze competitors (competitors skill)
3. WRITE → Create content (copywriting skill)
4. OPTIMIZE → Technical SEO (seo-audit skill)
5. SCHEMA → Add structured data (schema skill)
6. DISTRIBUTE → Promote content (social skill)
```

### Paid Acquisition Workflow

```
1. STRATEGY → Plan campaigns (ads skill)
2. CREATIVE → Design ads (ad-creative skill)
3. COPY → Write ad copy (copywriting skill)
4. LANDING → Optimize landing page (cro skill)
5. TRACK → Set up analytics (analytics skill)
6. TEST → A/B test creatives (ab-testing skill)
```

## Quick Reference

### Copywriting Frameworks

| Framework | Structure | Best For |
|-----------|-----------|----------|
| AIDA | Attention → Interest → Desire → Action | Ads, landing pages |
| PAS | Problem → Agitate → Solve | Pain-point pages |
| FAB | Features → Advantages → Benefits | Product pages |
| StoryBrand | Character → Problem → Guide → Plan → Call to Action | Homepage |
| Before-After-Bridge | Before state → After state → How to get there | Transformation pages |

### SEO Quick Wins

- [ ] Title tags under 60 characters with primary keyword
- [ ] Meta descriptions under 160 characters with CTA
- [ ] H1 matches search intent
- [ ] Internal links to related content
- [ ] Images with alt text and descriptive filenames
- [ ] Schema markup for rich snippets
- [ ] Page speed under 3 seconds
- [ ] Mobile-friendly design
- [ ] HTTPS enabled
- [ ] XML sitemap submitted

### CRO Checklist

- [ ] Clear headline matching visitor intent
- [ ] Single primary CTA above the fold
- [ ] Social proof (testimonials, logos, numbers)
- [ ] Risk reversal (free trial, guarantee, refund)
- [ ] Urgency or scarcity (limited time, limited spots)
- [ ] Form fields minimized (only ask what's needed)
- [ ] Mobile-optimized layout
- [ ] Fast page load time
- [ ] A/B test running or planned

## Common Rationalizations

| Rationalization | Reality |
|---|---|
| "Marketing is just creative" | Modern marketing is data-driven and systematic. Every decision should be measurable. |
| "We need more traffic" | More traffic without conversion optimization wastes money. Fix leaks first. |
| "Copy doesn't matter" | Copy is the interface between your product and your customer's brain. It matters enormously. |
| "SEO takes too long" | SEO compounds. Start now, benefit forever. |
| "A/B tests are too small" | Even small tests (100 conversions) can reveal meaningful insights. |
| "We can't compete with big brands" | Niche focus and better positioning beat broad appeal. |

## Red Flags

- No clear value proposition on the homepage
- Multiple competing CTAs on one page
- No analytics or tracking set up
- Copy focused on features not benefits
- No social proof or trust signals
- Forms asking for too much information
- No mobile optimization
- No email capture for non-converting visitors
- Pricing hidden or hard to find
- No competitor comparison or alternative pages

## Verification

Before launching any marketing campaign:
- [ ] Target audience clearly defined
- [ ] Value proposition tested with real users
- [ ] Tracking and analytics configured
- [ ] Landing page optimized for conversion
- [ ] A/B test planned or running
- [ ] Follow-up sequence prepared
- [ ] Budget and timeline set
- [ ] Success metrics defined
- [ ] Competitor positioning understood
- [ ] Risk mitigation planned
