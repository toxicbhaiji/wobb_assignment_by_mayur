# Anti-Slop Design Constraints

> Mechanism: the script only extracts each **bolded title** and injects it into the search result; the dash-led explanation is for maintenance only.

## 5 Hard Design Constraints (must be followed when writing DESIGN.md)

1. **No large flat color fills, including Hero/Banner/upload/action/input zones; Primary/Accent only as small focal accents** — large areas use neutrals, textures, imagery, or layering
2. **Do not use rounded-corner cards / thin outlines / generic drop shadows / left color bars / gradient backgrounds as the core visual signature** — these are auxiliary only
3. **The core visual signature must come from typography, texture/material, image treatment, localized light/shadow, or icon language** — the signature must be memorable and codeable
4. **No emoji icons, no CSS-faked logos, no system default display fonts** — use proper icon/font assets
5. **No style mash-ups; non-grayscale color roles ≤ 4; no looping decorative animations** — pick one direction and commit
